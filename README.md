# Advent of Code 2025

My solutions for [Advent of Code 2025](https://adventofcode.com/2025) challenges.

Advent of Code is an annual coding event with daily programming puzzles throughout December. This repository contains my solutions implemented in various programming languages.

## Running the Solutions

### JavaScript Solutions

```bash
node day-X-part-Y.js < input.txt
```

Or pipe input directly:
```bash
echo "your input" | node day-X-part-Y.js
```

### C++ Solutions

Compile and run:
```bash
g++ day-X-part-Y.cpp -o day-X-part-Y
./day-X-part-Y < input.txt
```
